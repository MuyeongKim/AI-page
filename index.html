Stay-Up

프로그램 소개

 이 프로그램은 딥러닝 기반의 AI 엔진(YOLO)을 사용하여 이미지와 동영상에서 객체를 빠르고 정확하게 탐지합니다. 

 구조출동(실종자 수색)시 실시간영상 및 사진을 쉽게 분석할 수 있도록 제작되었습니다.

릴리스 노트

업데이트 내역과 개선 사항입니다:

  • v24.12.14 - 탐지된 위치(사진GPS값 추출), 탐지종료시 지도에 자동으로 탐지위치표시 기능 추가
                     - 객체탐지모델 추가(yolov8x, VisDrone, fire는 예정)
                     - 기본장치를 GPU로 변경(GPU없을시 자동으로 CPU로 변경)객체 탐지 정확도 향상 및 새로운 기능 추가 (2024년 12월).
  • v24.12.01 - 진행률 창 표시, 객체탐지모델 추가
                     - 캡쳐보드,영상에도 사람만탐지 적용
  • v24.11.20 - 인증창 추가, 유효기간 연장 코드 추가
                     - '사람' 객체가 감지될 경우 팝업창으로 알림 기능 코드추가.
                     - 감지된 파일과 원본 파일을 별도 폴더로 복사하는 기능 코드추가. = 사진선별작업 시간을 줄여줌
                     - 탐지에 사람(person)만 탐지되도록 설정코드 추가                  
                     - 탐지에 걸린 시간 팝업에 표시 코드추가
  • v24.10.31 - 용량 최적화를 통해 프로그램 효율성 향상.
                     - 파일 선택기 개선으로 사용자 편의성 강화.
  • v24.05.13 - GUI 방식으로 재코딩하여 사용자 친화적 인터페이스 제공.
                     - 사진, 영상, 캡처보드 선택 기능 추가.
                     - 옵션 값(임계값, 장치 선택 등) 설정 가능하도록 개선.
  • v23.12.12 - 캡처보드를 활용한 실시간 객체 탐지 기능 구현.
                     -  Visual Studio Code 예제 코드를 기반으로 개발.
