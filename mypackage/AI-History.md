# AI 객체탐지 프로그램 히스토리

## 2026.04.09 (버전 V26.04.09)
- 버전 표기를 `26.04.09` 기준으로 통일했다.
- `check_version.py`의 현재 버전과 `latest_version.json`의 최신 버전 정보를 동기화했다.
- 메인 실행 경로에서 `Ui_MainWindow.run_app()` 진입점을 복구했다.
- 결과 표시 이후 앱이 다시 시작되던 잘못된 흐름을 정리했다.
- 모던 GUI 창 제목과 UI 리소스의 버전 문자열을 `V26.04.09`로 맞췄다.

## 2025.12.16 (버전 V25.1216 - Performance & UX)
- YOLO 탐지 로직을 `QThread` 기반 워커로 분리해 UI 멈춤 현상을 줄였다.
- 실시간 탐지 창 크기 조절 흐름을 개선했다.
- 탐지 처리 코드를 리팩터링해 유지보수성을 높였다.

## 2025.08.10 (버전 V25.0810 - Modern GUI)
- 모던 GUI 디자인과 Material Design 기반 스타일링을 적용했다.
- 버튼, 입력 영역, 카드형 섹션 등 UI 전반을 재구성했다.
- PySide6 호환성 대응용 fallback 실행 흐름을 추가했다.
- 메모리 모니터링과 FPS 버퍼 최적화를 도입했다.
